# SQL Data Analysis Project: Dairy Production Insights

## Overview
This project demonstrates SQL-based data analysis using multiple datasets including milk, cheese, coffee, honey, and yogurt production across U.S. states.

The project is divided into two parts:
- Part 1: Foundational SQL queries and data exploration
- Part 2: Business-driven analysis answering real-world stakeholder questions

## Objectives
- Analyze production trends across different commodities
- Identify high-performing states
- Compare production changes over time
- Support decision-making with data insights

---

## Part 1: SQL Fundamentals & Exploration

This section focuses on building core SQL skills:

- Total milk production (2023)
- Coffee production totals (2015)
- Average honey production (2022)
- State lookup and ANSI mapping
- Maximum yogurt production
- Cross-table analysis using JOINs
- Subqueries and conditional logic

Key SQL Concepts Used:
- SUM(), AVG(), MAX()
- GROUP BY
- CASE statements
- Subqueries
- JOINs

---

## Part 2: Business Analysis

This section simulates real-world business questions:

### Key Questions Answered:
- Which states had high cheese production in April 2023?
- How has coffee production changed over time?
- Which states are missing production data?
- What is the relationship between honey and coffee production?
- Which states produce across multiple commodities?

### Advanced SQL Techniques:
- Data cleaning using CAST and REPLACE
- LEFT JOIN for missing data analysis
- EXISTS for validation queries
- Multi-table filtering
- Aggregation across conditions

---

## Key Insights
- Identified high-production states for targeted market strategies
- Detected missing data across states for reporting gaps
- Compared multi-year production trends
- Linked production across commodities for strategic planning

## Tools Used
- SQL (SQLite / MySQL style queries)
- Data aggregation and transformation techniques
